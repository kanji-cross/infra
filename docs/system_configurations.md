```uml
@startuml
skinparam shadowing false
left to right direction
database 熟語DB {
  component 熟語table [
    熟語名
    頻出度（正確にはポピュラー具合、難易度にリンク）
  ]
}

folder 問題集 {
  artifact	問題 [
    id
    難易度
    サイズ
    問題(json?)
    抜かれた文字
  ]
}

[熟語インサータ] -> [mecab-ipadic-NEologd]: 熟語の一覧を取り出す
[熟語インサータ] -le-> [検索エンジン？]: 熟語の頻出度を調べる 
[熟語インサータ] -up-> 熟語DB

熟語DB <-up- [問題作成サーバ]: 難易度に応じた特定の頻出度以上の熟語を取り出す
note top {
  特定のサイズの問題を作成し
  いくつかの漢字を抜き出す
}

[問題作成サーバ] .up.> 問題集: リリースのタイミングで生成。追加のみで更新はしない。

actor クライアント

クライアント -> 問題集: レベル1から100まで解放されていく的な
クライアント -do-> [ヴァリデータ]: 解答を提出
[ヴァリデータ] -ri-> 熟語DB: 構成する熟語の存在をチェック

クライアント -le-> [問題作成サーバ] : ランダムな問題が欲しい人用(優先度低め)

@enduml
```
